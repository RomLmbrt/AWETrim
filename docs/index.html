How to add your own photos or plots

Place your image files inside the img/ folder, then edit content.js. For each block, change the image value to your file path and update the caption.

"ekf-awe": {
  title: "EKF-AWE Experimental Reconstruction",
  image: "img/ekf_reconstruction.png",
  caption: "Example reconstructed wind speed and kite states."
}
